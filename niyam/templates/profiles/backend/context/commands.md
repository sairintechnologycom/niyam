# Available Commands

These commands are available as slash commands in Claude Code (e.g., `/implement`).

## /implement

Plan and implement a feature or change using test-driven development.

**Usage:** `/implement <description of what to build>`

## /review

Review recent changes for correctness, security, and code quality.

**Usage:** `/review`

## /ship

Validate all changes, generate evidence, and prepare for merge.

**Usage:** `/ship`

## /context-refresh

Re-scan the repository and update project context.

**Usage:** `/context-refresh`
