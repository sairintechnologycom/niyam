# Requirement: Hello Command

Add a new `hello` command to the Niyam CLI.

## Feature Specifications

1. The command should be accessible via `niyam hello`.
2. It should accept an optional `--name` argument.
3. It should print "Hello, {name}!" if name is provided, otherwise "Hello, Niyam Developer!".
4. The implementation should follow the existing CLI structure using Typer.
5. Add unit tests for the new command.
